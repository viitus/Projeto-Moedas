# Ordem recomendada de calibração

1- Fixar a câmera (não mover mais).

2- Ajustar a iluminação (luz constante, sem sombras).

3- Medir o raio das moedas e calibrar:
 - TAMANHO_GRANDE_MIN
 - TAMANHO_MEDIO_MIN
 - TAMANHO_PEQUENO_MIN

4- Ajustar:
 - HOUGH_PARAM2

5- Ajustar:
 - HOUGH_MIN_DIST

6- Ajustar:
 - COR_PRATA_S_MAX

7- Ajustar:
 - SATURACAO_1_REAL_MIN
 - SATURACAO_1_REAL_MAX
